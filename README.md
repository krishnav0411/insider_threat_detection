AI-based Insider Threat Detection System

The Insider Threat Detection System is a cybersecurity-focused machine learning project designed to identify potential threats originating from within an organization by analyzing user behavior and system activity.

The system monitors actions such as login patterns, file access, and data transfers, and applies anomaly detection techniques to flag suspicious or unusual behavior.

It is built using Python, leveraging libraries like Pandas, NumPy, and Scikit-learn for data processing and modeling. A simple and interactive user interface is developed using Streamlit, allowing users to upload data, visualize patterns, and view detected anomalies easily.

The project focuses on preprocessing activity logs and applying models such as Isolation Forest or Local Outlier Factor to detect deviations from normal behavior, making it a practical and scalable approach for early insider threat detection.
